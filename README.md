# Проект на Vite + Gulp

**Vite** — сборка и разработка
**Gulp** — генерация шрифтов
**Squoosh** — сжатие изображений

## Установка

```bash
npm install
```

## Скрипты

### Локалка:

```bash
npm run dev
```

### Сборка

```bash
npm run build
```

### Конвертация шрифтов из ttf в woff/woff2

```bash
npm run gulp
```
